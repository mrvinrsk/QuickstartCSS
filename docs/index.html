Getting started

    To get started with QuickstartCSS you need to install NodeJS, to be able to use the npm command, if you haven't already. When you have NodeJS installed, you can simply run the following command: 

    npm i quickstartcss

    While reading this documentation you may notice that some styles need JavaScript to work. If so include jQuery by adding:

    <script src="https://cdn.jsdelivr.net/npm/jquery@3/dist/jquery.min.js"></script>

    to your html <head>, followed by the same tag with the quickstart.js file as src attribute.
